# Как открыть сайт (HTML-версию) на своём компьютере

Инструкция для папки **history_teacher_app** — помощник учителя истории Алтайского края.

---

## Самый простой способ (рекомендуется)

1. Откройте папку на диске:
   ```
   …\Приложение для учителя Истории по Истории Алтайского края\history_teacher_app
   ```

2. **Дважды щёлкните** файл **`ЗАПУСК_HTML.bat`**.

3. Подождите несколько секунд. Откроется браузер.

4. В адресной строке браузера:

   - **Главная (лендинг):** http://localhost:8080  
   - **Приложение:** http://localhost:8080/app.html  

   Если браузер не открылся сам — вставьте ссылку в Chrome, Edge или Firefox.

5. **Не закрывайте** чёрное окно (консоль), пока пользуетесь сайтом.

6. Чтобы выйти: закройте чёрное окно или нажмите в нём **Ctrl+C**.

---

## Что вы увидите на сайте

- Вкладки **«Тема из банка»** и **«Своя тема»**.
- Кнопка **«Создать готовый урок»** → текст конспекта.
- **«Скачать TXT»** — сохранить урок на компьютер.
- **«Печать»** — распечатать конспект.

Три темы в банке:

1. Алтай: земля, люди, история  
2. Как появился Алтайский край  
3. Освоение Алтая русскими людьми  

---

## Две версии программы — не путайте ссылки

| Файл запуска      | Ссылка в браузере              | Возможности                          |
|-------------------|--------------------------------|--------------------------------------|
| **ЗАПУСК_HTML.bat** | **http://localhost:8080**      | HTML, без Streamlit, вставка текста вручную |
| **ЗАПУСК.bat**      | **http://localhost:8501**      | Полная версия, загрузка текста по ссылке с сайта |

Обе ссылки работают **только на вашем компьютере**, пока запущено чёрное окно.  
**Нельзя** отправить `localhost` учителю в другом городе — у него не откроется.

## Если сайт не открывается

### «Python не найден»

1. Установите Python: https://www.python.org/downloads/  
2. При установке включите галочку **Add python.exe to PATH**.  
3. Снова запустите **ЗАПУСК_HTML.bat**.

### Страница пустая или «Не удалось загрузить банк тем»

- Не открывайте `index.html` двойным щелчком.  
- Запускайте только через **ЗАПУСК_HTML.bat** и адрес **http://localhost:8080**.

### Порт 8080 занят

Закройте другие копии программы (другие чёрные окна) и запустите bat снова.

### Нужна загрузка статьи по ссылке с интернета

Используйте полную версию: **ЗАПУСК.bat** → **http://localhost:8501**.

## Открыть без bat (вручную)

Если привычнее командная строка:

1. Откройте **PowerShell**.
2. Выполните (подставьте свой путь к папке):

```powershell
cd "C:\Users\VIP\Videos\1.Проекты Курсор\Приложение для учителя Истории по Истории Алтайского края\history_teacher_app\html"
python -m http.server 8080
```

3. В браузере откройте: **http://localhost:8080**

---

## Передать учителю

Передайте **всю папку** `history_teacher_app` (ZIP или флешка).  
Учитель у себя запускает **ЗАПУСК_HTML.bat** или **ЗАПУСК.bat** — не ссылку из вашего браузера.
