# HTML-версия — документация

## Назначение

Автономный веб-интерфейс для подготовки конспектов уроков по истории Алтая и Алтайского края.  
Работает в браузере без Streamlit.

## Файлы

| Файл | Назначение |
|------|------------|
| `index.html` | Разметка страницы, комментарии в начале файла |
| `css/styles.css` | Стили, адаптивная вёрстка, печать |
| `js/lesson-builder.js` | Функции `buildLessonFromBank`, `buildCustomLesson` |
| `js/app.js` | Формы, переключение режимов, загрузка JSON, скачивание |
| `js/lesson-bank.js` | Резервный банк (краткие тексты), если JSON недоступен |
| `data/lesson-bank.json` | Полный банк тем (основной источник) |

## Страницы

| URL | Файл | Назначение |
|-----|------|------------|
| `/` | `index.html` | Лендинг: приветствие, инструкция |
| `/app.html` | `app.html` | Рабочее приложение |

## Запуск

См. **`../КАК_ОТКРЫТЬ_САЙТ.md`**.

Кратко: `ЗАПУСК_HTML.bat` → **http://localhost:8080**

## Ограничения HTML

- Нет автоматической загрузки HTML-страниц по URL (CORS). Текст вставляется вручную.
- Для загрузки по ссылке — версия Streamlit (`ЗАПУСК.bat`).

## Синхронизация с Python

После правок `LESSON_BANK` в `app.py`:

```bash
python export_lesson_bank.py
```

Обновит `html/js/lesson-bank.js`. Файл `data/lesson-bank.json` правьте вручную или скопируйте из экспорта.
