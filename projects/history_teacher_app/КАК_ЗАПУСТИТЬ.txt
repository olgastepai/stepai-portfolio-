СПОСОБ 1 (САМЫЙ ПРОСТОЙ)
────────────────────────
1. Убедитесь, что на компьютере установлен Python.
   Скачать: https://www.python.org/downloads/
   При установке обязательно поставьте галочку «Add python.exe to PATH».

2. Дважды щёлкните по файлу:  ЗАПУСК.bat

3. Подождите 1–2 минуты (первый раз устанавливаются библиотеки).

4. Откроется браузер с приложением. Не закрывайте чёрное окно,
   пока работаете — через него запущена программа.

   Ссылка в браузере (только на ЭТОМ компьютере):
   http://localhost:8501


HTML-ВЕРСИЯ (сайт в браузере, без Streamlit)
──────────────────────────────────────────
Подробная инструкция:  КАК_ОТКРЫТЬ_САЙТ.md  (в этой же папке)

1. Дважды щёлкните:  ЗАПУСК_HTML.bat
2. Ссылка в браузере:  http://localhost:8080
3. Не закрывайте чёрное окно, пока работаете.

Отличие: в HTML текст с сайта вставляется вручную.
Загрузка по ссылке — в ЗАПУСК.bat (http://localhost:8501).


СПОСОБ 2 (ЧЕРЕЗ КОМАНДНУЮ СТРОКУ)
─────────────────────────────────
1. Откройте PowerShell.

2. Выполните по очереди (скопируйте целиком):

   cd "ПУТЬ_К_ПАПКЕ\history_teacher_app"

   pip install -r requirements.txt

   streamlit run app.py

3. В браузере откройте адрес, который покажет программа
   (обычно http://localhost:8501).


КАК ПОЛЬЗОВАТЬСЯ
────────────────
• «Тема из банка» — готовые уроки по трём темам.
• «Своя тема + источники» — введите тему, вставьте ссылку,
  нажмите «Загрузить текст по ссылке», проверьте текст,
  затем «Создать урок по своей теме».
• Кнопка «Скачать урок в TXT» — сохранить конспект на компьютер.


КАК ПЕРЕДАТЬ УЧИТЕЛЮ
────────────────────
1. Скопируйте на флешку или в архив ZIP всю папку:
   history_teacher_app

2. Внутри должны быть файлы:
   app.py
   requirements.txt
   ЗАПУСК.bat
   ЗАПУСК_В_СЕТИ.bat
   КАК_ЗАПУСТИТЬ.txt

3. На компьютере учителя — установить Python (один раз),
   затем запускать через ЗАПУСК.bat.


ПОЧЕМУ ССЫЛКА ИЗ БРАУЗЕРА НЕ ОТКРЫВАЕТСЯ У ДРУГОГО ЧЕЛОВЕКА
────────────────────────────────────────────────────────────
Адрес вроде http://localhost:8501 означает «этот же компьютер».
Другой телефон или ноутбук не видит ваш localhost — для них это
не существующий адрес.

Что делать:

• Учительу в другом городе / без вашего компьютера — передайте
  всю папку history_teacher_app (флешка, ZIP) и пусть запустит
  ЗАПУСК.bat у себя. Это надёжный способ.

• Показать с телефона или планшета в той же Wi-Fi (дом, школа):
  1) На вашем ПК запустите ЗАПУСК_В_СЕТИ.bat (не обычный ЗАПУСК.bat).
  2) В чёрном окне появятся адреса вида http://192.168.1.5:8501.
  3) Отправьте учителю именно такой адрес (с цифрами, не localhost).
  4) Ваш компьютер и чёрное окно должны быть включены всё время.

• Через интернет «как сайт» без своего сервера — из коробки не
  получится; нужен хостинг (Streamlit Cloud и т.п.) — отдельная настройка.


ЕСЛИ НЕ ЗАПУСКАЕТСЯ
───────────────────
• «python не является внутренней командой» — переустановите Python
  с галочкой Add to PATH.

• Страница по ссылке не загружается — откройте сайт в браузере,
  выделите текст, Ctrl+C и вставьте в поле «Материал из источников».

• Закройте приложение: в чёрном окне нажмите Ctrl+C или просто
  закройте окно.
